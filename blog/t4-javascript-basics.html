How does JavaScript compare to HTML and CSS?

Javascript is a programming language used to make your web page interactive by listening to user actions and make changes on the page

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order of how statements are executed. Loops are used to determine whether a st of statements need to be repeated.

Example:

  1. Add toothpaste to toothbrush
  2. brush your teeth
  3. all teeth brushed?
    1. No - go back to step #2
    2. Yes - exit
  4. Wash your mouth
  5. Wash the toothbrush

Explain the difference between accessing data from arrays and objects.

Arrays: access and modify data using indexes (numeric)

Objects: access and modify data using properties (can be anything)

Explain what functions are and why they are useful.

Functions are a collection code and it is useful because you can call the function multiple times without rewriting the whole code
