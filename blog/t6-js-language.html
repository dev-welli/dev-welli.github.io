T6: JavaScript Language

In the JavaScript language, describe the use cases for parentheses ()

Parentheses indicates a function. Function is a block of code that when called or invoked, perform task. Imagine a doc that does the same trick only when asked.

A function can take an input called 'arguments' which need to be enclosed in parentheses and separated by commas.

In the JavaScript language, describe the use cases for brackets []

Brackets indicate arrays. Arrays is a collection of data or values where each value has an 'index' or location associated with it.

Brackets are used to store the arrays values and its indexes

For example:
var flavors = ['strawberry', 'chocolate', 'vanilla']
In the above, the array is called 'flavors' which has 3 values and the second value, indicated by flavor[1] is chocolate. Note: Array's index starts with 0.

In the JavaScript language, describe the use cases for braces {}

The curly braces has many uses in Javascript:

  • Defining a function
  • Creating a looping or conditional statements
  • Defining a JavaScript object

In the JavaScript language, describe the use cases for single quotes '' or double quotes ""

They are used to enclosed a string and they are interchangeables as long as it is consistent. If the string is opened with a single quote then it needs to be closed with single quote too.
