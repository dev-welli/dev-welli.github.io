What is grid based design?

Grid based design is breaking up the web page into a series of columns and rows using CSS so the layout can be easily defined.
Contents of the page can then be positioned on a grid that you want

Whats all the hype about responsive webpages?

Nowadays there are many devices with different screen witdhs/resolutions such as mobile phones, tablets, laptops, desktops etc. By designing a responsive web page, its content will automatically adjust to the screen's resolution for better user experience.

What is semantic structure?

The use of HTML tags to define elements within it. For example, h is for header and h1 is used for the most prominent one i.e. h1 is larger than h2. Other examples are p for paragraph, img for image and br for break line
